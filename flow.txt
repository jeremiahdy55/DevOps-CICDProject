1. Have the microservice(s) code in the Github monorepo
2. Create the terraform scripts.
3. Create an EC2 instance for terraform to run on.
4. Configure AWS credentials using command: "aws configure" in the terraform EC2 instance
4a. *Need to use IAM roles to generate access key and secret access key*
5. Upload the .tf scripts into the terraform EC2 instance, then run:
5a. "terraform init"
5b. "terraform plan"
5c. "terraform apply"
6. Log into Jenkins
6a. sudo cat /var/lib/jenkins/secrets/initialAdminPassword
6b. Install suggested plugins for Jenkins 
7. Configure Github and Docker credentials for Jenkins
7a. Manage Jenkins > Credneitals > global (add credentials from dropdown)
7b. DOCKER - Username: Insert docker hub Username
7c. DOCKER - password: Insert docker hub personal access token
7d. DOCKER - ID: variable name for Jenkins to access this information in scripts and jobs

7e. GITHUB - Username: Insert Github hub Username
7f. GITHUB - password: Insert Github hub personal access token
7g. GITHUB - ID: variable name for Jenkins to access this information in scripts and jobs

sudo apt update

curl -fsSL https://apt.releases.hashicorp.com/gpg | sudo gpg --dearmor -o /usr/share/keyrings/hashicorp-archive-keyring.gpg
echo "deb [signed-by=/usr/share/keyrings/hashicorp-archive-keyring.gpg] https://apt.releases.hashicorp.com $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/hashicorp.list
sudo apt update
sudo apt install -y terraform
